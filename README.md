# Eightey Portfolio

## Replacing the logo
A placeholder logo is at `assets/logo-placeholder.svg`.
To use your own logo:
1. Add your image to the `assets/` folder (e.g. `logo.png`).
2. In `index.html`, update both `<img>`/`<link>` references:
   - `<link rel="icon" href="assets/logo-placeholder.svg" ...>`
   - `<img src="assets/logo-placeholder.svg" alt="Eightey logo" class="logo-img" />`
   to point to your new file.
